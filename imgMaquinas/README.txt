CARPETA DE IMÁGENES DE MÁQUINAS
=================================

Esta carpeta contiene las fotografías de las máquinas del sistema.

ESTRUCTURA:
-----------
- no-maquina.png : Imagen por defecto cuando no hay foto
- 1.jpg/png/gif  : Imagen de la máquina con ID 1
- 2.jpg/png/gif  : Imagen de la máquina con ID 2
- etc...

IMPORTANTE:
-----------
1. Coloca una imagen llamada "no-maquina.png" como imagen por defecto
2. Las imágenes de las máquinas se nombran automáticamente con su ID
3. Formatos permitidos: PNG, JPG, JPEG, GIF
4. Tamaño máximo: 5MB por imagen
5. Esta carpeta se crea automáticamente al subir la primera imagen

PERMISOS:
---------
Asegúrate de que esta carpeta tenga permisos de escritura (777 o 755)
para que el sistema pueda guardar las imágenes.

En Linux/Mac:
chmod 755 imgMaquinas

En Windows:
Dar permisos de escritura a la carpeta desde las propiedades.
